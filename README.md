# Estudos de Análise de Dados com RStudio

Este repositório reúne meus estudos e práticas de **Análise de Dados** utilizando a linguagem **R** e o ambiente **RStudio**.  
O objetivo é registrar meu aprendizado, compartilhar códigos e explorar conceitos essenciais de estatística, visualização e manipulação de dados.

---

##Objetivos

- Aprender os fundamentos da linguagem **R**
- Entender o fluxo básico de uma **análise de dados**
- Explorar **visualizações** com `ggplot2`
- Manipular dados com o pacote `dplyr`
- Praticar **importação e limpeza de dados**
- Criar pequenos projetos e estudos de caso
